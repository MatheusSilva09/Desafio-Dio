Utilizando tags básicas do HTML

A estrutura básica do HTML.

  • HTML: É a tag que inicia e finaliza o documento HTML, ela mostra ao navegador que tudo que está entre a abertura de tag <HTML> e fechamento de tag </HTML>, trata-se de um texto em HTML.
  • A tag HTML consiste em duas partes. São elas, <HEAD> e <BODY>.
    1. A tag <HEAD> é o cabeçalho da página, essa tag é responsável por ser a primeira coisa que o usuário irá visualizar antes de a página carregar. Nela pode ser adicionado um <TITLE> que é o título da sua página que é mostrado na aba do navegador.
    2. A tag <BODY> é o corpo da página, aquilo que vai ser mostrado ao usuário dentro da página HTML.
  • TAGS: É uma ferramenta utilizada para informar ao navegador que tipo de estrutura está sendo construída. As tags possuem atributos que podem ser globais ou específicas para cada Tag.
  • TEXTOS: Textos podem ser divididos em títulos que vão da tag <H1> até a tag <H6> .
  • LISTAS ORDENADAS E NÃO ORDENADAS: As <OL> e <UL> são tags utilizadas para criar listas, sejam elas em uma sequência numérica ou apenas listadas em sequência sem numeração. As listas podem ser utilizadas também para criar menus interativos.
  • LINKS: Link significa ligação, e no HTML é utilizada a tag <A> para adicionar um link.
